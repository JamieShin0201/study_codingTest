### 2주차 
  - 다음 모임 30일 10시 - 1시 예정
  - 1주차 주제 ( 스택, 큐 )로 추가 문제 풀이 ( 백준 - 1874(수열), 11866(요세푸스) )

  - git flow 관리도 겸하여 학습의 의미로 각자의 브랜치를 나누어 관리 및 병합한다.
  - 전체적인 개념 공부 및 예제를 푼 후, 난이도를 높여서 다시 복습한다.
